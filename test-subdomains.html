🧪 Squareft Subdomain Testing

📋 Setup Instructions:

  1. Add these lines to your /etc/hosts file:
    2. 127.0.0.1 squareft.local
127.0.0.1 app.squareft.local
127.0.0.1 portal.squareft.local
  2. Start your Next.js dev server: npm run dev
  3. Test the links below:

🔗 Test Links:

🏠 Landing Page (squareft.local:3000) 🏢 Main App (app.squareft.local:3000) 👥 Tenant Portal (portal.squareft.local:3000)

🔍 What to Check:

  • ✅ Landing page shows at squareft.local:3000
  • ✅ Main app routes to /app/* when accessed via app.squareft.local:3000
  • ✅ Tenant portal routes to /portal/* when accessed via portal.squareft.local:3000
  • ✅ Navigation links work between subdomains
  • ✅ Middleware correctly rewrites URLs

🐛 Troubleshooting:

  • Can't access subdomains? Check your hosts file and restart browser
  • 404 errors? Make sure Next.js dev server is running
  • Middleware not working? Check browser console for errors
